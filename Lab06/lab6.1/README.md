# Lab 6.1 — Multi-Page Application with React Router (TypeScript)

## Objective
Build a simple “Student Portal” multi-page app using react-router-dom v6 with:
- Home (/)
- Courses (/courses)
- About (/about)
- NotFound (*) for unmatched URLs  
All pages share a common Layout with navigation and footer, and child routes render via Outlet.

## Tech Stack
- Vite + React + TypeScript
- react-router-dom v6

## How to Run
```bash
npm install
npm run dev
